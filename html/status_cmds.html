Read RSSI

Read the Received Signal Strength Indication (RSSI) from the Controller.

Command

OCF

Command Parameters

Return Parameters

READ_RSSI

0x0005

Handle
Status
Handle
RSSI

Parameters

Handle, size = 2 octets

Value(s)

Parameter Description

0x0000 to 0x0EFF

Connection handle for which the RSSI should be read.

Status, size = 1 octet

Value(s)

Parameter Description

0x00

0x01 to 0xFF

Handle, size = 2 octets

Value(s)

Parameter Description

RSSI, size = 1 octet

Value(s)

Parameter Description

0x00 to 0xFF
RSSI (signed value).
Range: -127dBm to 20dBm, 127 if invalid.
