Read Local Supported Commands

Reads the list of HCI commands supported by the local Controller.

Command

OCF

Command Parameters

Return Parameters

READ_LOCAL_SUP_CMDS

0x0002
Status
Supported_Commands

Parameters

None.

Return Parameters

Status, size = 1 octet

Value(s)

Parameter Description

0x00

Command completed successfully.

0x01 to 0xFF

Command failed.
Supported_Commands, size = 64 octets

Value(s)

Parameter Description

N/A
Bit mask for each HCI command, such that when
Bit X = 1: Command indicated by Bit X is supported.
Bit X = 0: Command indicated by Bit X is not supported.
